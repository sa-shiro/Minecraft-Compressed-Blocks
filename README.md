# Compressed Blocks Mod 1.14.4
 
Compressed Blocks mod for Minecraft 1.14.4 

Compress your Blocks to make more free space (up to **3.486.784.101** Blocks in just ***one sinlge Block***

1.15.x version will be released under the "master" branch

---
To compile this project you either need the [Multiproject setup](https://github.com/sa-shiro/MDK_1.14.4) or manually install and setup the forge 1.14.4 MDK
